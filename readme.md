# Martingale charts

Provides Area, Bar, Line, and Pie charts to Martingale UI by wrapping up the existing recharts library.

## Install

Available once we opensource everything

```sh
yarn add martingale-charts
```

## Running tests

```
yarn test
```

## Creating a build

```
yarn compile
```
